# Weather Telegram Bot on aiogram

Современный Telegram-бот на Python и aiogram 3.x с красивым меню, эмодзи-интерфейсом и прогнозом погоды для выбранного города.

## Этот проект демонстрирует создание современного Telegram-бота на базе aiogram 3.x с асинхронной архитектурой, модульной структурой и интеграцией внешнего API. 

Бот красиво оформлен, выводит прогноз погоды в удобном формате 
легко настраивается под любой город через .env без изменения основного кода.

## Что умеет бот

- показывает приветственное сообщение через `/start`
- выводит список команд через `/help`
- отправляет прогноз погоды через `/weather`
- поддерживает кнопки с эмодзи для более приятного интерфейса
- получает погоду асинхронно через Open-Meteo API
- хранит токен и настройки города в `.env`
- имеет чистую модульную структуру проекта

## Сильные стороны проекта

- асинхронная архитектура на aiogram 3.x
- понятная разбивка по папкам и файлам
- быстрый старт даже для новичка
- безопасное хранение токена вне основного кода
- адаптивное меню с крупными кнопками
- красивый и аккуратный текст ответов

## Запуск

- Вставь свой токен в .env.
- Установи зависимости командой pip install -r requirements.txt.
- Запусти файл main.py.
